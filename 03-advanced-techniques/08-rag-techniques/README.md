# RAG Techniques

Move beyond basic vector similarity search. Learn hybrid retrieval, knowledge graphs, contextual chunking, and agentic RAG patterns that handle complex, multi-hop queries.

## 🎯 What You'll Learn

- Implement hybrid search combining keyword (BM25) and semantic retrieval
- Use re-ranking to filter and prioritize retrieved context
- Build GraphRAG pipelines with knowledge graphs for relationship-aware retrieval
- Design agentic RAG where the agent decides when and what to retrieve
